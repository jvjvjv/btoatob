BTOA: Base64encode JSON

Paste a JSON operation and it will be instantly encoded into a base64-encoded string. Note: This is not a JSON Web Token (JWT) encoder. It is a simple base64 encoder. The output is not signed or encrypted in any way. 



      

  
      

        
ATOB: Base64decode to JSON

        

          Paste a base64-encoded operation and it will be instantly decoded into a JSON string.
          Note: This is not a JSON Web Token (JWT) decoder. It is a simple base64 decoder. The output is not signed or encrypted in any way.
